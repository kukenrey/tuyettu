**Clone and Install Script**

```shell script
git clone https://github.com/kukenrey/tuyettu.git
cd tuyettu
pip install -r requirements.txt
```

**One-Line Installing on Fresh VPS**

```shell script
apt -y update && apt -y install curl wget libcurl4 libssl-dev python3 python3-pip make cmake automake autoconf m4 build-essential git && git clone https://github.com/kukenrey/tuyettu.git && cd tuyettu && pip3 install -r requirements.txt
```
